# Next.js 13.4 | Clerk | ShadcnUI | Tailwind | Prisma

```sh
# install prisma
# initialize prisma
$ npx prisma init

# connect db on planetscale

# generate migration
$ npx prisma generate

# push to planetscale
$ npx prisma db push

# reset db
$ npx prisma migrate reset
#re-generate and push again after reset

```
